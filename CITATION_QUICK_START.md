## URL Verification Checklist

For each URL you collect:
- [ ] Does it work (200 status)?
- [ ] Is it official (.gov or .org)?
- [ ] Is content current?
- [ ] Is it freely accessible?
- [ ] Is it HTTPS?
- [ ] Is URL stable (not session-specific)?

---

## After Collection

1. Copy URLs from this document into:
   - CITATION_NEEDS_DETAILED.md (for specific content items)
   - CITATION_SOURCES_BY_TYPE.md (to mark as collected)

2. Update source files:
   - Edit `src/content/topics/*.ts`
   - Edit `src/content/resources/*.ts`
   - Update governance blocks with new sources

3. Verify:
   - Run `npm run build`
   - Check governance validation passes
   - Test URLs work
